Project #4 Portfolio Redisigns

Description

The project for class 5

Although in previous classes we made a simple version of our own portfolio website from a template with fixed designs, now in class we are making redesigns of our portfolios. I wanted to make them look like cat patterns, so I drew some wireframes for my concept.

Design Process

My boyfriend have three cats: a typical chinese cat, a ragdoll cat, and a scottish flod cat. I really love them, my family doesn't let me have any pets so I always wished to have a cat with me, and these tree cats really cured me. That's why I decied to incopreate them in my design. I wanted to make them look flat and like a color pattern instead of looking like cat, so I didn't draw the background into the shape of cats.

Reflections

I would have made the cat pattern loop more smoothly if I have time.
